# Spark Logistic Regression Text Classification

## Overview  
This notebook implements logistic regression to classify text documents using Apache Spark. It covers feature extraction using TF-IDF, feature standardization, and model training with gradient descent to identify Australian court case documents from a mixed corpus.

## Features  
- Data loading and preprocessing with Spark RDDs  
- Creation of a reference dictionary for frequent words  
- TF-IDF feature extraction  
- Feature standardization (mean centering and normalization)  
- Preparation for logistic regression model training  
- Scalable processing for large text datasets  

## Tech Stack  
- Python  
- Jupyter Notebook / Google Colab  
- Apache Spark (PySpark)  
- NumPy  

## How to Use  
1. Clone the repository.  
2. Run the notebook in Google Colab or Jupyter with PySpark installed.  
3. Progress through each cell to load data, create features, and standardize them.  
4. Use the standardized features for logistic regression training and evaluation.
